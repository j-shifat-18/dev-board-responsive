Question-1: What are the different ways to select an element in the DOM?

getElementById, getElementsByClassName, querySelector, querySelectorAll, getElementsByTagName

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Using innerText, we can get and set the text inside an element, but using innerHTML, we can get and set all the content inside the element, including HTML tags. textContent is similar to innerText but includes hidden text and does not process HTML.

Question-3: What is event delegation in the DOM?

Event delegation means adding an event listener to a parent element instead of each child. The event bubbles up from children to the parent, so we can handle multiple elements with one event.

Question-4: What is event bubbling in the DOM?

Event bubbling means when an event happens on an element, it first runs on that element and then moves up to its parent and then further up until it reaches the root.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement("elementName"). To add, use parentElement.appendChild(newElement). To remove, use parentElement.removeChild(childElement).
